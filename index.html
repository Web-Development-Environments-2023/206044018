The differences between the webs:


Web 1.0 is the initial version and the websites were static, not dynamic - "you get what you see" and with basic, non-interactive design.
Web 2.0 is the time that the websites were more collaborative, social and dynamic. The main change was the possibility of sharing content by users.
Web 3.0 is using AI and Natural language processing therefor the websites more intelligent. The network tries to find the best information,
not only to present information but also to process it.

use case of search for remedy:
web 1.0 - Need to know where the site of drugs is through a portal, then search the page of specific drug and at the page need physically look for what is needed.
web 2.0 - Write in the search engine the desired drug, then you have to physically go through the results and check which ones are relevant.
web 3.0 - Write in the search engine the desired drug, the search engine is with intelligent AI that understands the context.
In addition there were summarize of some of the pages. Therefore, the user knows that the returned documents are relevant to what he is looking for and it is easier for him to decide.
